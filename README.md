# Simple Python Password Manager

A simple password manager written in Python.  
This project allows you to **store, search, and delete passwords** for different websites.  

> ⚠️ Currently, passwords are stored in plain text. Future updates will add encryption and a master password.

## Features
- Add new website login entries (website name, username, password)
- Delete entries with fuzzy search for typos
- Saves data to a local JSON file (`Password_Data.json`)
- CLI-based interface (simple and easy to use)

## Future Plans
- Encrypt passwords with a master password
- Add a GUI for easier usage
- Auto-generate strong passwords
- Copy passwords to clipboard
- Backup & restore password database
- Multi-user support
- Logging and undo features
